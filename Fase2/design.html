Flex box

A raíz de la pandemia del COVID-19 la educación virtual ha venido a suplir la necesidad de la presencialidad, siendo usado por todas las instituciones educativas, esto ha traído como resultado la necesidad de elaborar recursos educativos digitales para dinamizar las estrategias de aprendizaje. Una estrategia la constituyen los Objetos virtuales de aprendizaje. Flexbox es un diseño unidimensional que puede colocar elementos horizontal o verticalmente (pero no ambos). Un elemento que usa flexbox como diseño se conoce como contenedor flexible , y los elementos dentro de él son elementos flexibles.

CSS Grid

CSS son las siglas en inglés para «hojas de estilo en cascada» (cascading style sheets). Básicamente, es un lenguaje que maneja el diseño y presentación de las páginas web, es decir, cómo lucen cuando un visitante las visita. Funciona junto al lenguaje HTML, que se encarga del contenido básico de las páginas. Se les denomina hojas de estilo «en cascada» porque puedes tener varias hojas y una de ellas con las propiedades heredadas (o «en cascada») de otra.

El diseño de una página web, y de todo un sitio, necesita un lenguaje en común para que los navegadores puedan interpretarlo y mostrarlo a las personas de la forma correcta. Pero también para que otros diseñadores hagan las modificaciones necesarias para su optimización. CSS Grid es un modelo de maquetación CSS en base a una rejilla, es uno de los nuevos valores de HTML5 que podemos dar a la propiedad display de un elemento, y que nos permite maquetar las páginas web de manera más fácil, a traves de filas y columnas.
